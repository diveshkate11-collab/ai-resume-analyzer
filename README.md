# AI Resume Copilot

An AI-powered resume analysis platform that extracts resume information, calculates ATS scores, recommends suitable job roles, and provides actionable feedback to improve resumes.

---

## 🚀 Current Features

- PDF Resume Parser
- DOCX Resume Parser
- Resume Text Cleaner
- Contact Information Extraction
- Skills Extraction
- Education Extraction
- Experience Extraction
- Unified Resume Parser

---

## 🛠️ Tech Stack

### Backend
- Python
- FastAPI *(Coming Soon)*

### AI / NLP
- Regular Expressions (Regex)
- Custom Resume Parsing

### Data
- Text-based Skills Database
